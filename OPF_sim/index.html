How to Use This Tool
1. Setup Tasks: Click "Add Task" or "Import Tasks Data".
↳ CSV Format: 4 columns (Task Name, Opt, ML, Pess). Row 1 is ignored.
2. Order Them: Type numbers into the "Seq" boxes and hit "Sort Order" below.
3. Real Data: Click the REC button next to any task to open the data collector.
4. Simulate: Set iterations (e.g., 5000) and click RUN to generate your capacity report.
